STITEK Bontang

  • Step 1: Membuat halaman web sederhana
    • Penggunaan tag HTML dasar
    • Penggunaan tag <head> dan <body>
    • Penggunaan tag <title>
  • Step 2: Elemen HTML Dasar
    • Penggunaan tag <h1> hingga <h6>
    • Penggunaan tag <p> untuk paragraf
    • Penggunaan tag <ul> dan <li> untuk daftar
  • Step 3: Menambahkan gambar dan tautan
    • Penggunaan tag <img> untuk menampilkan gambar
    • Penggunaan atribut src dan alt
    • Penggunaan tag <a> untuk membuat tautan
  • Step 4: Penggunaan Text Formatting
    • Penggunaan tag <strong> untuk teks tebal
    • Penggunaan tag <em> untuk teks miring
    • Penggunaan tag <u> untuk teks bergaris bawah
  • Step 5: Membuat Formulir Pendaftaran dan table
    • Penggunaan tag <form> untuk membuat formulir
    • Penggunaan tag <input> untuk input teks, email, dan telepon
    • Penggunaan tag <select> untuk dropdown pilihan
    • Penggunaan tag <table> untuk membuat tabel
